## Install dependencies
``npm install``

## Client
If you want run client: 
``npm run start-game``

if build:
``npm run build-game``

## AI Server
If you want run AI server in browser to validation boxes movement and actions: 
``npm run start-ai``
 and open browser and serverPreview.html file.


if build and run in nodeJs:
``npm run build-ai``
